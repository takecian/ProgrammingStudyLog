どうなんだろうな話。

昨日の続き、企業の売り上げ全体のうち海外の売上比率がどれだけ占めるか、
他の企業はどうなんだろうと気になって調べてみた。

大手電機の海外売上比率(2007年度)

ソニー：74.4%
東芝：49.4%
日立：40.5%
松下：49.3%
シャープ：51.2%

こうやってみると電機の中では比率高いんだなーと。
ちなみに 任天堂：66.5% とか。

他の業界の売り上げ比率見て思ったのが
自動車業界の海外売上比率の軒並み高いこと。

トヨタ：74.3%
ホンダ：84.8%
日産：78.9%
マツダ：72.7%

もうすでにトヨタや日産は減産の発表とかしてたから
(北米工場の閉鎖とか、高級車の売り上げ鈍化をうけて生産縮小)
この業界も今回のリーマンショックで大打撃うけてんだろな。
