公開している友人一覧

やり手とは

こういう人だと

いう見本



よっしゃ週末キタコレな話。

なんだかんだで今週結局毎日残業してたがな。

と思ってみたけど残業時間数えてみたら今週で約１０時間。
数字でみたらたいしたことない？

まぁそんなこと忘れて明日明後日と楽しんだったるわ。


その前に今週印象に残ったことの話。

昨日の朝いきなり課長さんに抜き打ちテストをやられ、
それから課の先輩(チューターではない)と俺と課長さんとで
週一回朝に勉強会をしよう、ということになった。

課長なんてアホみたいに忙しいはずなのに
俺の勉強会なんてやってくれて資料も用意してくれたりして、
いつ自分の仕事やってんだろと思ってたんだけど。

すごいなーやり手だなーと。


その勉強会の場で課長さんが、
「あ、タケユキ君この勉強会のことチューターには言ったらだめだから」

俺「？」

課長さん「だってタケユキ君が(チューターの)知らないところで
力をつけてきている、それに気付いたらチューターも焦るでしょ？(笑)」

鬼だ、鬼がいた。


土日はほんと鬼のいぬ間に洗濯したりますか。
雨っぽいのが気になるけど。
