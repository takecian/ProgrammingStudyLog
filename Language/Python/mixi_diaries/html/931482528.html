忘れない

どんなときでも

笑うこと



一日一回笑えるとなんとかなるんかねな話。

エンジニアってのはたいていが武器を持ってます。

それがメカ屋や電気屋なんかだったら工具とかになるんだろうけど
ソフト屋にとっての武器ってのはソフトウェアや
PC上の何かしらのファイルだったりするわけで。


たいていそういう武器をUSBメモリとかに入れてるんよね、
問題が起きた時に「こういう時にはこのソフトを使って解析する」
とかってことをしたり。


ソフト屋のなかでもグラフィック系の人は、

"コマ落ち(映像がカクカクなるやつ)を目で見つけやすい動画ファイル"や、

"ちゃんと色が表示されているかを確認しやすい画像ファイル"なんかを
持ってたりするそうで。


今日うちの課の試作機でディスプレイの色の表示が
おかしいんじゃないかって話が出て、

ある人に、

「色合いを検証するための画像ファイルをください」

って言ってもらってきて、

開いてみたら、

知らないグラビアアイドルの写真キターーーーーーーー。


馬鹿じゃねえのｗ


先輩が、

「ちゃんと色出てますね、それにしてもこの子かわいいね。
　もう一回○○さんのとこ行って、他にも画像ないか聞いてきて」

とか言い出すし。


「(今修羅場だから)でもそんなこと言ってられる状況じゃないですよね？」

って言ったら、

「馬鹿野郎！それはそれ！これはこれ！
　ずっと張り詰めたままだと持たんぞ」

と。


んで他の画像もあったらくれって言いに行ったら、

「他のはどっかいったよ、○○ってとこから落としてきたやつだけど。
　てか色あいを見る画像なんて普通の写真ならなんでもいいんだよ」

と呆れ気味に言われたorz


それを先輩に報告したら、

「よし、じゃあタケユキ君、ちょっとネットで探してきてくれるか」

おいおい。

まぁ修羅場じゃこういう冗談言いながらじゃないと無理なんね。


先輩は明日も休日出勤するらしいけど、
俺は三連休の始まり、始まり。
