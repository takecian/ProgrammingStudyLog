公開している友人一覧

ストレスの

発散法は

散財で




ふと気付いたこと、
それは自分なりのストレス発散法が散財だったってこと。

今日はちょっと用事で天王寺の方まで行ってたんだけど、
帰りの電車の中でなぜか強く思ったのが、
「あーー金使いてえぇぇぇええぇ！！」
だった。

それから頭の中は何に金を使ってやろうかってこと
考えてたらあっという間に家ついてた。

プラン１、「マンガ買いまくる」
最近ちょっと興味あるマンガをそろえてみる。
「キングダム」とか「サタン」とか。
昔懐かしの「世紀末リーダー伝たけし！」が完全版になってるから
それを揃えてみるのもありかもしれん。
終わりがどうなったか知らんし。

プラン２、「雑貨買いまくる」
ビレッジヴァンガード行って買いまくる。
とにかく買いまくる。
ちょっと興味持ったシャレた人が持ってそうな本とか、
あとなんか面白そうなやつもとにかく買ってみる。

プラン３、「DVD買いまくる」
カップラーメンのCMの月にいる主人公が
地球見に行くみたいな「フリーダム」っていうやつ、
あれ実はCMだけじゃなくてDVDにもなってて
ちゃんと細かいストーリーもあって２巻まで出てるから買ってみる。


こういうこと考えてストレス発散。
ストレスたまるんだからいいじゃない、人間だもの。
