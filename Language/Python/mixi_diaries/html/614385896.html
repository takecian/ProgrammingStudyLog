マンガ読み

買わなくなって

みたりして




少し食生活の話。


今週号(＆先週号)のスピリッツの「美味しんぼ」。
なんか知らんけど"食の安全"をテーマにした話で。

・清涼飲料水(炭酸飲料等)にはペットボトル１本あたり
　砂糖40g(シュガースティック6本分)ぐらい入ってる。

・オレンジジュースは香料と化学調味料と砂糖だけで
　オレンジの味を作ってる

・カップラーメンとかのスープはほとんど化学調味料だけで作られてる
　
とかとか。

とりあえず興味ある人はコンビニに並んでる
スピリッツで「美味しんぼ」読んでみるといいかも。


どっかで見た内容だなーと思ってたらニコニコ動画の
「ケミカルクッキング」ってやつでやってたやつと似てるんだ。

参考：http://www.nicovideo.jp/watch/sm1053539
(アカウントがないと見れなかったりします)


こういう話をどこまで信じたらいいのかは分からんのだけど
実際言われてみたら怖いもんで。

確かに普通にラーメンの作り方とかを考えたら
１００円で作れるようなものではものではないはず＞袋麺。

オレンジジュースも材料考えたら
あんなに安く作れんと確かに思うわけで。


コンビニ弁当とかのご飯は見た目がよく見えるようにツヤがでる
化学調味料入れて炊いてるとか言われると、
なかなか手が伸びにくくなるもので。

しかし一人暮らしの人間にはコンビニ弁当とかは便利なわけで。
でもちょっと食べたくない気持ちがでてきたので
できることなら自炊でもしようかと今日思ってみたり。

それと同時に自分の料理のレパートリーのなさに愕然とするわけで。
自分で作れる料理をパッと考えてみるも、

・焼そば
・スパゲッティ
・お好み焼き(広島風)
*******追記*****************
そいやドリアも作れたわ。
*******追記終わり***********

胸を張れるのはお好み焼きだけか。
明日から少しずつ頑張ってみようかと思ってみたり。

と思って今日学校帰りにスーパー寄って思ったこと、
「もやしめっちゃ安い」

当分主食になりそうです。
