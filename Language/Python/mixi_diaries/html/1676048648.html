公開している友人一覧

しっかりと

肩までつかって

百数え



懐かしいな話。

知ってる方は知ってると思いますが、
長野県は温泉の宝庫でして。

会社から車でいける範囲にかなりの数の温泉＆銭湯があります。


話変わって、我が家はただいまプロパンガスでして。

住んでる家がオール電化や都市ガスじゃない人は
知ってると思うけど、

このプロパンガス、くっっそ高いのね。


先月のガス代とか１万円超えるくらい。


まぁ水道代を入れても、
1回あたりのお風呂にかかるお金は、
さすがに銭湯行くより(400-800円、場所によって違う)安いんだろうけど、

それでも1回あたり数百円はかかるレベル。


それならいっそのこと温泉入ったほうがいいやんけ、
と今週から会社帰りにちょくちょく温泉に行ってます。

今日はここ。瑞祥。
http://www4.ocn.ne.jp/~zuisyo/matsumoto.html

上山田温泉(松本の北のほう)からお湯を持ってきてるらしい。

1回650円と高いんだけど、サウナもあるし、
今後通うだろうと思って、回数券も買ってみた。

＃ちなみに瑞祥は来週は入浴料500円キャンペーンやってるらしいですよ。


それでお風呂に入ってたら、隣から

「い〜ち、にー」

と子どもが大きな声で数を数える声。


あーお父さんに、

「１００数えて上がりなさい」

って言われたパターンだな、と。


明日は白馬に乗り込むため、今日は早めの就寝。

一週間おつかれでした。
