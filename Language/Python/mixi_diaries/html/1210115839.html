公開している友人一覧

学祭の

売り子のような

気分かね



別にナンパしてるわけじゃないです製品説明をしてるだけな話。

今日の戦果：T75,A750,T75,T75,T75,X95の合計６台。


販売実習に来てるけど売り上げ台数にノルマがあるわけじゃないので
お客さんに声かけられない限りは
基本的にこっちから声をかけないスタンスでいるんだけども。

お客さんの後ろで、いつでも聞けますよ的雰囲気で待っていて。


そこにふと後ろからかわいい女の子が「すいません」と。

「テレビとレコーダーを見に来たんだけど
　全然分からないので教えてください」


と。

ｷﾀ━━(━(━(-( ( (ﾟ∀ﾟ) ) )-)━)━) ━━ !!!!!


ですな。


聞いてみれば地デジ環境はあるんだけど
持ってるのがアナログのテレビデオで、買い替えを考えてる、と。


一番簡単なパターンですがな。

＃中には地デジ環境がない(アンテナ工事が必要)、
＃ケーブルテレビで自分の家がどうなってるかよく分からん、とかめんどいのがいろいろいる。


よく言うけど接客ってどんなに忙しくても一度に相手するのは１人(１グループ)なわけで。


そこで、


「じゃあレコーダーもテレビも全部説明しますよ」


とその子の接客時間およそ１時間。



これは楽しかった。

こういう子ばっかりだったらいいのに。


やっぱりテレビやレコーダーは家族連れが多いね。

ビデオカメラは妊婦が多いらしいけど。

デジカメ・ウォークマンあたりは若い子が来そうだけども。


さて、あと１６回。
