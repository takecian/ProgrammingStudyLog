２Ｇありゃ

そりゃあ十分

動くじゃろ



というわけで今日は朝からインテックス大阪で
関西ビジネスショウのうちの研究科のブースにいました。
出展関係者のタグつけてパンフレット配って研究説明の一日。

ビジネスショウってのになんで大学がブース出してんだろ。

ほとんど技術系の会社の人だったろうけど
おっちゃん相手に説明説明。

暇になったころを見計らって他社のブース見学。
マイクロソフトが Windows Vista を公開してたからいじってみた。

Office2007も入ってて感想としてはXP + Mac みたいな感じ。
けっこうおしゃれになったんじゃないかと。

しかしデモ機のスペック見てみたらビビった。
ノートＰＣなのにメモリを２Ｇ積んでるんですか。

ハイスペックが要求されるVistaだけど(最低必要メモリが512MB)
２Ｇのメモリ積んでサクサク動くとか言うのは
ちょっとずるいんじゃないかと思う。

夕方まで説明マシーンと化して17時撤収。
立ちっぱなしは疲れたー。
