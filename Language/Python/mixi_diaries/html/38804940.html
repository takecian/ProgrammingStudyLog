今回は恋愛バトンだとかどうとか。

■Ｑ１■
理想の恋人像を教えてください。

・俺を手のひらで躍らせるようなもしくは俺が手のひらで
　踊りたくなるようなタイプの人
(必ずしもプライドは高くなくていい)

・たまに俺にそっけない態度をとる人

・でも最後は俺を受け入れてくれる人

■Ｑ２■
恋人選び、見た目と性格を重視する割合は？

正確に何対何とかってのは分からんけども
見た目がよければ性格は何割り増しかよく見えてしまうのは
男としては仕方のないことかと。

まぁ性格がよければ見た目も何割り増しか良く見えるんだろうけど。

でもだいたい俺は見た目重視と人は言います。

■Ｑ３■
今日は一日好きな人と一緒。
あなたの考えるデートプランを教えてください。

バイクで迎えてに行って昼ごはんはポムの樹。
メニュー選びに１０分悩む。
午後にちょっと走って公園とかでひなたぼっこ。
夕方映画観て、夜に居酒屋で晩ご飯。
とりあえずお酒はがまん。
帰りにちょっと寄り道して夜景見て送り届ける。


俺が家に帰ってくるまでに今日は楽しかったよメールが
来てるかどうかでその日のデートの勝ち負けを判断。

■Ｑ４■
好きな人と、はじめて二人でカラオケに行くことになりました。
さぁ、どんな曲を歌いますか？
また、相手には何を歌ってもらいたい？

自分の好きな曲。
相手がどんな曲を歌ってもあまり興味がないのでなんでも。
カラオケってのは自分が歌って楽しい場所だと思ってます。

■Ｑ５■
夜の遊園地、はじめて二人で観覧車に乗りました。
ドキドキクライマックス！
手をつなぐ？つながない？

つなぐに決まってる。てかあわよくばキスしようとすると思う。

■Ｑ６■
楽しいデートの時間はあっという間。
いつの間にか、終電がなくなっていました！
そんなとき、あなたならどうする？？
そして、相手になんて言う？

「今夜は帰りたくない・・」って言う。

そう言ったら過去に「うっさい帰れや！」と言われた経験あり。

■Ｑ７■
相手をかなり気に入ったあなた。
告白は自分からする？
相手からされるのを待つ？

気に入った時点で体から好き好きオーラが出るのですでにバレてる。
それでも相手が逃げてなかったら言うと思う。

たまに好き好きオーラ出してても逃げないのに
そっけない態度をする人がいて迷う、迷ってる。

■Ｑ８■
ずばり、いま好きな人、気になる人がいますか？

恥ずかしくて言えない。

↑こういう質問をはぐらかすやつの真意ってのは
えてしてyesだったりすることが多い。

■Ｑ９■
その人は、mixi内に生息しますか？

いない。俺を捨てた人なら生息してる。


■Ｑ１０■

さんま、オグ、ガク、ダニエル、カリカリーンさん。
