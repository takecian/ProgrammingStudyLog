明日から

東京二泊

行ってきます



というわけで月曜に会社の配属面談があるので
フライングして明日の昼に東京へ。

当初の予定では月曜に配属面談含め懇親会もやるから
３時間くらいかかるぜって言われてたのに
昨日電話があって、
・予定が変わって配属面談のみです
・なので３０分くらいで終わります
とのこと。

俺とは懇親する気はないですかそうですかorz


場所もインターシティだったのに本社に来いと。
なんかインターシティに縁がないなー。

配属面談・・。
結局何も話したいことがない状態で突入しそうだ。
まぁなんとかなるか。


とりあえず１８日に大阪戻ってくる予定なんだけど
できることならもうちょっと日程がずれてたら
１９日に東京であるオトコク祭(木村カエラライブ)行けたのにorz

んでまぁせっかくなので東京にいる友達と会おうと
声かけてみた結果、時間は短いけどけっこう会えそう。


一応お土産も持っていこうかと思ったけど、
お土産ってだいたい１つ６００円くらい。

今のとこ計５人と会う予定。
６００×５＝３０００

・・けっこうかかることない？


とりあえず泊めてもらう友達のは買うとして、
関西人の人には大阪土産はいらんか。

まぁ広島の友達は年末に会うからいらんか。


お土産予算は６００円。
これでいこう。

まぁどうせ八つ橋だけど。

せっかくだしまゆさんの売ってるお店の八つ橋を
持っていけたらよかったけど時間がないので無理ぽいorz



しかし東京ってなんか今楽しいことあんの？

せっかくだから友達に会うだけじゃなくて観光もしたいけど
・別に買い物したいわけじゃない
・ジブリ美術館は予約制だから今からは無理
・SMAP STORE にもあんまり興味ない


まぁ行ってから考えるか。


じゃあの。
