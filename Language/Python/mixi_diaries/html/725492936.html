やめてくれ

その一言は

余計じゃろ



おかんの話。

今日広島からおかんがうちへやってきました。
引越しの下準備に処分するものとか
今度俺と入れ違いで大阪に引っ越してくる知り合いの子に
電子レンジをあげるとかどうとか。

案の定部屋きて、

「あんたようこんなにマンガ集めたもんじゃねえ。
どうすんこれ？
お父さんがワンピース読みたい言うとったよ」

と。


その辺はほっといてくれ。
そして親父あきらめてくれ。


んで、

「そういや今日って阪大入試かなんかなんじゃろ？
千里中央駅らへんでマンションのチラシたくさん配りょーたわ。
通るたびにいろんな人から渡されるけん、

『けっこうです、うちは出て行く方ですから』

ってちゃんと言うたったわアハハハ」


イヤイヤ断るのにうちの事情まで言わんでいいじゃろ。

親って一言いらんこと言うよね。
その場におらんでよかったと心底思う。


それにしても引越しって大変じゃな。
市役所で転出手続きやって、郵便局に転送依頼だして
あとなんかいろいろ。

めんどくさ。

そして研究室の退去作業もやらねば。
明日も研究室に行ってきます。


昨日今日とラジオでオールナイトニッポン４０周年スペシャルやってそうで。
ビートたけしとかゆずとか所ジョージとか。

タモリのラジオとか聴いてた世代じゃないけどおもしろそうで。
