右左

正しいとかは

ないけども


帰ってくると風呂が沸いてるっていいねな話。

寮には大浴場って言って共用の風呂場があるんだけども。
軽く泳げるんじゃね？くらいの広さでして。

会社から帰ってきたら沸いてるんよね、
掃除のおばさんがやってくれてるんだろうけども。


デュアルディスプレイの話。

１台のパソコンに２台のモニタをつなげて使うってやつ。

俺は研究室の頃からデュアルディスプレイで使い始めて
今は家の PC もデュアルで使ってたりします。

参考：



んで、今日会社で１台モニタの余りが出たってんで
もらって会社もデュアルにしようと
セッティングしてたわけですが。


デュアルディスプレイの場合メインで使うモニタ(プライマリ)と
サブモニタ(セカンダリ)って分けられてまして。

windows のスタートボタンとかがある方がメインモニタ。
要するに主に作業する方のモニタ。


んで研究室のころからメインを右に、サブを左に置いてから
会社でもそうしようかと思ったら、
配線の長さの関係でうまくいかずorz

四苦八苦してたら先輩が、
「そんな無理せず普通にサブモニタを右に置いたらいいじゃん」


普通に？

俺の常識とは逆ですが。


んで他の人に聞いてみたところ
メインを右、サブを左派が２人、
メインを左、サブを右派が２人。

どっちが正しいとかじゃないんだけど
サブモニタを左にして使うもんだと自然に思ってたから驚いた。

あなたはどっち派？

って聞いてみてもデュアルな人ってそんなおらんかorz


微妙に風邪気味。

寮はまぁ閉鎖空間みたいなもんだから
誰か風邪ひくと移っていくわな。

今日は２２時には寝よう。
