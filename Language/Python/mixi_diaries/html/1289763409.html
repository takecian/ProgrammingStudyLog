月曜日

月９が今から

最終回




北川景子がかわいいからという理由で見てみる。

ストーリーが全然分からんけど
前回までで山Ｐと離れ離れになることになって、
今回の最後でまたくっつくという流れかと。


北川景子確かに可愛いわ。



今日はバイクで軽く千葉の方へ。

養老渓谷あたり。







今回は動画用ではなく、静止画用のカメラを持っていきまして。



ひょーーー。

Nikonのデジタル一眼買ってしまったーー。

またしてもサッカーができるくらいの諭吉を放出。


デジタル一眼っていいね。

なんか写真を撮ってる感じがする。

あのシャッターが下りる音がいい。



しかしね、物欲ってすごいね。

もうこれで当分必要なものはないだろうと思ってたら、

デジタル一眼を買った
→いい写真を撮れるはず
→撮ったら見ねばならんはず

→プリンタが新たに必要、フォトフレームが新たに必要



無理だわ。貯金。


まぁ形に残ってるだけ、まだ多少マシか。
