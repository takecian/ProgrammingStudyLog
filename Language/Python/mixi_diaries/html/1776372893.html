一週間

旅行でなくて

出張で


アナハイムに行って来ましたな話。

アナハイム・・ロサンゼルスの南東にある街。ディズニーランドがある。
ロサンゼルスの近くにはハリウッドとかもある。


先輩の運転するレンタカーでハリウッド。


めっちゃ小さいけど、HOLLYWOODの看板。


ジョニーデップの手形・足形＠チャイニーズ・シアター。


サンタモニタの海岸。これは太平洋。


飯。やっぱりご飯は日本のがいい。


デパート。


マイクロソフト・ストア。日本では見たことないかも。


アップルストア。平日にもかかわらず大繁盛。




ディズニーランド。
こっちもランドとシーみたいな感じで、
ディズニーランドリゾートとカリフォルニア・アドベンチャーっていう
２つのテーマパークがあるらしい。

今回はディズニーランドリゾート。


一緒に行った会社の先輩。


カリブの海賊はお休みだったorz


シンデレラ城？


夜のショーもド派手な感じでして。



ここだけ書くと遊びに行ったんかい、と思えるけど
仕事しに行ってまして。

今のマイクロソフトの一番偉い人、スティーブ・バルマー。


Windows8 担当の偉い人、スティーブン・シノフスキー。


会場では飲み放題、食べ放題。


イベントの途中にはパーティも。


パーティでは RA RA RIOT っていう今年のフジロックに来たバンドがライブも。
マイクロソフトさんの力すげえ。


帰りに Windows8 が入った tablet を貰って来ました。
ホーム画面がこれ。


次の Windows では大分変更があるらしいですよ。


というわけで刺激になった一週間でございました、と。
