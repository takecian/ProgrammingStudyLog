SNS

どこもかしこも

SNS



というわけで使い分けをしないといけませんな話。

mixi をはじめ、

twitter,

facebook,

Google+

その他もろもろ、と

SNS、

いわゆる 自分の独り言を世界に発信サービス が

たくさん出てきたわけでして。



↑に出したメジャーどころはとりあえずアカウントは持ってるものの、

・全 SNS に対して同じことを書くようにする

・各 SNS(アカウント) で使い分けをする

どっちにするのがいいんでしょうか、と。


うちの親父が俺のツイッターを見ているらしい、という噂を
聞いて即非公開にしたのがいい例で、

どの SNS も「公開する」設定にしておくと誰でも見れるわけで。

誰に見られてるか分からんわけで。


会社の人(上司とか)が見れるような(facebookとか)ものに
悪口なんか書いた日には大事件だし。

まぁ書きませんけどね、基本的に。


こないだアプリの広告配信業者から「うちの広告使いませんか？」と
メールが来たけど、

その差出人の名前が変わった名前だったのでググッてみたら、
その人の facebook のページが出てきて、

趣味まで分かるとか。ゴルフはまってんだな、とか。


プライバシーなんてありませんな世界。


普及しすぎると実社会みたいな
気を遣わんといかんめんどくさいところまで出てくるわけですね。


ヘタなことはできんし、書けん。


ちょっとしんどいな。
