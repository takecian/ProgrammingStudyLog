公開している友人一覧

連休は

生き物たちと

ふれあい中



堪能しております、ゴールデンウィークな話。

昨日は上野動物園、
今日は八景島シーパラダイス。



上野ではやぎちゃんとランチをしてみたり。



シーパラでは打ち上げ花火をやってた。

明日は一休憩で品川近辺でのんびり。


やー、東京遊ぶとこたくさんあるわ。
