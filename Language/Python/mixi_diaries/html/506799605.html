本当の

自分の気持ち

伝えることは

難しい





ケータイでメールをする時の話。

最近は異なるキャリア間(ドコモ・KDDI間とか)でも
絵文字がほぼ表示されるようになったから
メールを送るときに絵文字を使う人は多くなってると思います。

俺は昔の絵文字がキャリア依存だった名残りとか、
そういうのでメールで使うのは基本顔文字。

こんなんとか (>_<) 、こんなんとか(*_*) 。
たまにパターンすくねぇよと突っ込まれたりするけど。

男同士で絵文字を多用するのもなんかなーと思うし
女の子に送る時はまぁ顔文字なり☆マークなりを
つけて送るようにしてたわけです。
ほぼ絵文字は使わない派。


でも中には「絵文字なんかめんどくせーよ」とか
メールするのめんどい、みたいな子はいるだろうと思ってるわけで。

俺の友達の中にもそういうタイプの子だよなーと
思う子がいるわけで。

そんな子とメールをする時はあえて絵文字はもちろん
顔文字とか☆マークでさえ使わずに
文末をあえて"。"で終わらせるように合わせてメールしてた。

就活が終わって報告メールを送る時もそんな感じで。


こないだその子と会う機会があって。
まぁなんでもない話をしながら飲んでたんだけど
ふとその子から、

・なんであんたのメールは絵文字が一個もないん？
・てかあたしとメールするのはイヤイヤしとんのか？

・連絡系のメールもしゃあなしで送っとんか？
・就活の報告に「おめでとう」って返信したのに
あたしにはそれの返事返さんかったくせに、
○○(共通の友達)には返事返したじゃろ。最悪かあんた。

みたいな話をされまして。


イヤイヤイヤイヤイイヤイヤイヤイヤイヤ。

俺はあんたに合わせてメールを送ってたつもりなのに。
特別にってほどじゃないけど意識してメールをしてたのに。

いろいろ話をしてみると俺の想像とは180度反対で
メールは男でも絵文字を使いまくれ派だったようで。

人間は見た目やその雰囲気から判断するのは
必ずしも正解ではないってのを身をもって知った。



でも実際どうなん？
男でもメールでは絵文字を使う方がいいもん？

もちろん自分が好きなように送るのが一番なのは分かるけど
今回みたいに知らず知らずのうちにあらぬ誤解を招くんだったら
メールの送り方を考え直した方がいいんだろうし。


というわけでこれからは俺のケータイメールでも
絵文字が登場するようになります

使い時が分からんので余計に気持ちが
伝わらんのじゃなかろうかという心配も軽くあるけど

まずは練習練習
・・メールの練習ってなんかおっさんみたいじゃな
