公開している友人一覧

頭では

分かっていても

心では



結局は独占欲の塊なんですねな話。


昨日は休みになるはずが
おとといの退社時に急な仕事が舞い込んできたため急遽出社することに。

仕事ってのは基本キャッチボールでして。

自分が作業して相手にパスして、
相手が作業して自分に、または他の人にパスして、
そうやって仕事は進んでいくんだけども。

おとといの退社時に俺にボールが回ってきまして。
俺がボールを持ったまま休みに入るとまずいってので
急遽昨日出社することにしたら、

係長＆課長から「休みじゃなかったの？」的な突っ込みのメールが。

休むって言ってた日を部下がいきなり出社に変更するんだから
何かあったのか気になる気持ちは分かる。

でも、もちろん俺だって好きで出社するわけじゃないわけで。
やらないといけないことができてないから出社するわけで。

気持ち的に余裕はあまりないわけで。


そういう時に係長から、

係長「なぜわざわざ休みをキャンセルして出社するのですか？
　　その理由を納得できるように教えてください。　※原文まま」

とかってメールされると、
「・・・めんどくさ！というかうざっ！」と思ったり。

って思って次にきてた課長からのメールを開くと、

課長「仕事はかどってないん？あまり無理せず。　※原文まま」


好みはあるんだろうけど、

係長からのメールを読んでやる気をなくして。
課長からのメールを読んで頑張ろうと思えて。


同じ内容のことなのに文章ひとつで気持ちって大きく変わるね、
余裕がない時は特に。



と話がそれてたので、本題。

前に、

「社会人になると、ある程度の年になると
　いい女(男)にはたいてい男(女)がいる」

理論を唱えていたんだけど、
それを超えた壁の向こうにもさらに壁があることに気付いた。

「社会人になると、ある程度の年になると
　誰にもいろんな過去がある」

理論。


自分と出会う前に恋をしてほしくないわけではないけど、

学生生活、出会う前の社会人生活、
そりゃ何かありますよ、いい女(男)ならとくに。


ふとしたしぐさ、何気ない会話からそういうのを
感じ取るのが男なわけで。

過去まで自分のものには出来ないってのは頭では分かっているけど、
心まではなかなか受け入れられないのもまた事実。


結婚する相手は学生時代から知ってる人だといいなという意見に
強く共感できる今日このごろ。


強くならんといかんな、と心に誓った、
そんな GW の始まり。

今日は半そででも寒くないくらいのいい天気。
心もさっさと晴れてくれませんかな。
