ちくしょうめ

あの女が

またやった



昨日は高校１‐４(通称元４)の忘年会だったわけです。

０次会はいつものごとく賀茂ボウルでボーリング。
最下位には自腹で一人プリクラの罰ゲーム。
大阪戻ったらスキャンして
ここで晒してやろうと思います。
いいよね？

今回の忘年会は一人勢い余って
元４じゃない子(がすがす)が参加してました。

男９人＋女１人という状況(アウェー)なのに
どホームのようにくつろぐがすがす。

そういう飲み会の場で財布見せたら
クレジットカードとかパクって、
「よし、今日はおごりで飲むぞ〜！」とかやるじゃないですか、冗談で。
そんで笑いながらカード返して終わり、みたいな。

飲んでる途中でがすがすにカード取られたわけですよ。
あ〜こいつうぜえな。
てかうぜえ通り越してもうだりぃ、みたいな。

それから俺が半分潰れかけて
がすがすが荷物持ってくれてたりしてたから、
あぁもうカードは戻しといてくれたんだと。

なんだかんだ言って空気読めるやつだしな。
いっつも帰省の時季節限定八ツ橋買って帰ってあげてるしな。

それで解散して、さっき
「八ツ橋早く食べろよ」
メールしたら、




「(あたし)カード持ってるよ」




はぁ・・！？


財布見たらカードがない。
さらにゆうちょのカードもないorz

それから俺は探したよ、
向かいのホーム、路地裏の窓、
そんなところにいるはずもないのに！！

チリンチリン・・じゃなかったカードがない寂しさを
紛らわすために行きずりの女とも寝たよ！！


でもカードの代わりにはならなかった・・。



話脱線しました、戻します。


というわけで高校の時に見知らぬ人を装って
メル友になろうよって罠を仕掛けてきたがすがすに
"また"はめられました。
↑当時その罠にもかかりました。

そんな風味で今年は過ぎていきそうです。
来年はもっと自分に誠実に、厳しく生きていければと思います。
それではみなさん、よいお年を。
