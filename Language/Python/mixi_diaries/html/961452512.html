ぼちぼちと

冬計画を

練ってみる



なんであんなにいい値段するのな話。

今日も昨日に引き続いて松本散策を。
と言っても普通に休みの日を満喫してただけですが。

ふらっと入った本屋で見かけた本、「阪急電車」。




どこかで聞いたことある本だと思って
手にとってみたけど、なんかいいね、この本。

参考：有川浩「阪急電車」
恋の始まり、別れの兆し、そして途中下車、
関西のローカル線を舞台に繰り広げられる片道わずか15分の物語。


話の舞台、阪急今津線(宝塚駅から今津駅まで)の
さまざまな乗客の視点で描かれる日常。

別にそれほど起伏がある物語ではなくて、
ありそうでなさそうで、でもやっぱりあるかもしれないような、
そんな話が綴られていく。

大阪にいたころもほとんど使ってはなかった路線だけども
阪急電車は宝塚線をよく使ってたせいか興味をひかれた。

少し力を抜きたい時に読みたい本。


今年の冬にスノーボードをやろうかと思ってまして。

スノボ自体はオグんとこにお邪魔した時の１回しかやったことないけど
楽しかったなーという記憶はあって。

んで今日ムラサキスポーツ寄ってボードとか見てたんだけど、


めっっっっっっっちゃ高いね、スノボの道具って。


あんな板で６万とか７万とか。

電子機器とか一切入ってないのに。なにが高いの？


ぱーっと見たけど、全部揃えようとしたら、
ボードで６万、バインディング＆ブーツで５万。
ウェアとかゴーグルとかでさらに６〜７万。

もろもろで２０万近くかかるんじゃねえの？これ。


いやいやいくらなんでもそんなには出せませんぜ、兄さん。
ボードやってる人どうやったの？教えて詳しい人。
