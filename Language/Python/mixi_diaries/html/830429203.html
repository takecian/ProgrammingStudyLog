二次情報

ずっとそれでは

ダメなんです



成長するとはの話。

の前に体重の話。

前日との体重変化が、
月：-0.2kg
火：-0.6kg
水：+0.2kg
木：-0.4kg
金：±0kg

こんな調子でございます。

寮の脱衣所に体重計があって
風呂上りに毎日乗ってるんだけど
毎日乗るってのはいいね、いい感じに意識する。


ジム行き始めてからだいたい -2kg ぐらいで。
まぁ体重よりも体力つけたいとこですが。

最初は笑えるくらい体力落ちてたけど
最近はそれなりになってきたのではないかと。

それでもまだまだだけど。



今日会社でミーティング。

俺の仕事量、というか短い期間に求められるアウトプットが
かなり増えたようでございまして。


というか締切が来週って何？


それで時間に追われながら仕事をしてますが、
先輩方からすれば俺はパソコンは素人みたいなものでして
ことあるごとに「すいませんうまくいかないんです助けてください」を。

幸いなことに課の人たちは優しくて、
「これをこうしたらできるよ」

教えてくれるけども。


それじゃダメなんですわ。

よく分からんけど、こうやったらできるらしい、
よく分からんけど、おまじないとしてこれしとけばいいらしい。


人から教えてもらった、二次情報だけで納得してたら
理解できないんです、成長しないんです。

自分で仕様書や文献を読んで、考えて、
なんでそうするのか、こうするとどうなのか、
それを理解しないとまた誰かに聞くだけですよ。


と課長さんから諭された。



がんばります、来週から。

というわけで、

土日だぜーーーーーーーーーーーーーーーーーー。
