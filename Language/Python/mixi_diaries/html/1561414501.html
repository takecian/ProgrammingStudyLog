まずいまずい

人間腐る

のは一瞬



というわけで学生みたいな終わった生活になってるお盆な話。

夜３時に寝て昼に起きるとか。


今年のお盆は実家に帰らず松本滞在にしたんだけど、
学生生活並に腐った感じになってしまって非常にまずい。


長野は昼は暑いけど夜は過ごしやすい(エアコン入らず)ので、
夜更かしが進行しつつある。


会社行ってる時は朝６時に起きて、
近所ジョギングして、それから会社という
素敵生活を送りつつあったのに。

もう一瞬で腐ったね。


まずいまずい。
