英語強化の夏になりそうな話。

うちの研究室は毎年夏にみんなで(教員含む)
TOEICを受けよう、みたいな行事があります。

しかも教授の娘まで参戦するおまけ付。

同い年とかなら負けてもいいやと思うも、
教授の娘、高一。

去年の中３の時点でTOEIC400ちょっとあったらしいから
今年は院生だれか倒されるんじゃないかって噂。

てか中３で400あるってどんな中３や。

なわけでちょっとがんばらんと。
しかもTOEIC500切った学生は国際会議に出席禁止とかに
しようとかって言ってるのを聞いた。

今んとこ査読の結果待ちの段階とは言え、
もし論文通ったら行く予定なんですけど、海外。

まぁ通らんくてもいいんだけど。
いや、通ってほしいですが。
