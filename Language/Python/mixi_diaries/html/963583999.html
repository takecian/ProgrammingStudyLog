世間への

アンテナ鋭く

立てておく



それはつまり食わず嫌いをしないということな話。

次期 Windows の名前がコードネームからそのまま決まったね。
Windows 7 だそうで。


mixi では日記を書くことがメインで他には
→に出ているニュース記事を読むくらいだったんだけども、

長野でホテル滞在するようになってから「mixi ミュージック」にも
手を出すようになりまして。

参考：mixi ミュージック・・簡単に言ったらただで音楽が聴けるやつ。
１：まずここをクリック↓
　http://music.mixi.jp/search_music.pl

２：ミュージックチャンネルの一覧から好きなジャンルを選ぶ


これだけ。
聴きたいアーティストの音楽が聴けるかは分からんけども
暇つぶしがてら音楽を聴きたい人には便利。


使い方とかもそんな難しいわけじゃないけど
現状に満足してると新しいサービスがあっても
めんどくさくなって手を出さないってこと多いよね。

ケータイの新機能とかもたぶんそう。

「メール・電話できればそれで十分」と俺も思ってたけど
それは言い換えると新しい機能を理解、使い方を学ぶのが面倒なのよね。


ケータイといえば最近は携帯端末の値段があがってて。

だからいっそのことスマートフォンに買い換えようかと思ってて。

参考：スマートフォン・・従来のケータイより高機能なケータイ。
iPhone もジャンル的にはスマートフォン。
メジャーなスマートフォンには windows mobile が搭載されてるものも多い。

でもケータイだから持ち歩いてて自慢したくなるようなものがいいなと。

となると注目がいくのはやっぱり XPERIA シリーズだよなと。



参考：XPERIA シリーズ・・ソニーエリクソンのスマートフォン。
　　　windows mobile 搭載。


他には一部で話題になってるアンドロイドケータイ。



参考：アンドロイド・・google が開発した携帯電話用 OS 。
　　　これが搭載されてる端末は google map や
　　　ストリートビューとか google が提供するサービスとの親和性が高いらしい。


この辺気になるところだけど問題があって。


今のとこ日本で発売する予定はないらしい。


日本の携帯市場は他と性質が違うせいですかね。


んで現実的な線では 台湾 HTC の touch pro かなーと。



これも windows mobile 搭載。


これはドコモとソフトバンクで出るらしいんだけど
まだ発売日未定の状態。


同じ携帯を長いこと使ってると、
メールアドレス変わったりするキャリアをまたぐ機種変更は
敬遠するようになるんで、というかなってるんで
ドコモがこれ出すまで機種変は我慢ですか。

携帯変えるのいつになることやら。


しかし日本はスマートフォンはまだまだマイナーみたいね。
アメリカだとたくさん出てるっぽいのに。


世間にアンテナ広げるのは大事だけど
知りすぎるとと寂しいことも多いから考えものか。
