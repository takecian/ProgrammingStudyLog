府知事選

橋下弁護士

当確で



現在RADWIMPS「オーダーメイド」高頻度再生中。


さっきテレビで速報やってたね。
というわけで今日は大阪府知事選挙でした。

細かい票差は知らんけど結局知名度で買った橋下弁護士が
府知事になるそうで。

まぁなるからには頑張って欲しいところ。
と言っても大阪にあとちょっとしかおらんのだけど。

てか引越しがいつになるんか分からんけど
大阪におるのってあと二ヶ月切ったんじゃね？


時間経つのはええな。



昨日ベップとかと話題になってたけどmixiの話。

mixi内ってけっこうな人間がいるもんで
けっこうな確率で昔の友達or知り合いって見つかるもんね。

でもマイミクになるような人間ってのは
たいていが実際に連絡取ってる人だったり
最近付き合いがあった人とかで、
そういう人はすでになってるわけで。

んで、

１：マイミクじゃない人から足跡がある
２：その足跡を辿っていくと知り合いだった
３：こっちから声かけようと思うも、
　　あっちが俺に足跡つけたのにメッセージもマイミク申請もしていない
４：あっちは俺と絡みたくないのではないだろうか
５：結局こっちからも足跡残しただけで何もしない

このパターンが多いわな。
またはこの逆。

自分が足跡残してあっちからのアクションを期待する、
いわゆる待ちの状態だったり。

でもあっちが↑の３・４を考えて何も言ってこなかったり。

そんで一人複雑な気分になったりするよね。

他にもマイミクにするにしてもどこまでをマイミクにするかって
基準は人それぞれだからまた難しい。

輪を広げすぎると好き勝手日記が書けんくなったりするし。
「助手の先生がおるからmixiで研究室の愚痴が書けん」って
言ってる人もいたし。

読んで(聞いて)もらいたい日記と読んでもらいたくない日記と
あったりするから吐き出したい時に吐き出せなかったり。

こういうの考えてmixiめんどくさってなる人多いらしいけど。


考えてみたらmixiだけに限らずネット上で
個人情報ってけっこう出てきたりするよね。

去年の冬かそこらに実家帰った時に親父に
「あんたの研究テーマの○○・・ってのはどんなんなんね？」
と言われたり。

俺の名前が珍しいってのもあるんだろうけど
自分の名前フルネームでグーグル検索すると出てくるね、
自分の出した論文だとか研究室のHPだとか。

まぁ下手なことはできませんな。
