Windows 種類多すぎ 殺す気か   多様性の話。 Windows。いわずと知れた Microsoft の OS。 ちょっと前に Windows Vista が出て、今度 Windows 7 が出るって話だけども。 そんな OS の移行期ってのは PC メーカーってのは大変で。どっちの OS にも対応せんといかんので労力が。 今回の Windows 7 も Vista みたいにたくさん Edition があるらしい。 Home Premium Professional Enterprise Home Basic Starter Ultimate こんな感じらしい。 まぁ日本の市場で出回るのは Home Premium とか Professional あたりなんだろうけど。 ↑にある中で見たことないような Edition の Enterprise は企業向け、ちなみに Starter ってのは新興国向け。 正規の OS を買う余裕のない人が多い国は海賊版(違法コピー)が蔓延してるっていう状況をよく思ってない Microsoft が新興国向けに提供する廉価版のOS。 ただ安い分低機能になってるとかいう話。 噂だとこの Starter Edition は一度にウィンドウを３つまでしか開けないんとか。 まぁ日本では関係ない話だとは思うけど。  んでさらに問題はこの OS の Edition とは別にあって。 32bit OS と 64 bit OS。 CPU が一度に扱うデータサイズが 32 bit から 64 bit に。  これが原因で 32 bit OS で動いてたソフトウェアが64bit OS では動かなくなることがあるっていう話があったけど、  今日実際に体験した。 そのせいで今日一日を無駄にした。  プログラムが動きません。 →なにかモジュールが足りてないんじゃないの？→足りてる →とりあえず再起動しようぜ→問題解決せず →どこかプログラム間違ってるんじゃないの？→そんなことない昔から使ってるやつなんです →じゃあそのマシンがおかしいんじゃないの？→じゃあ他のマシンで試してみよう→あれ？こっちのマシンだとちゃんと動く・・・ →そのマシン・・何 bit ？  →( ﾟдﾟ)ﾊｯ!　　←ここに至るまでに半日かかった。  もうね、種類なんて一種類でいいんですよ。 という感じで金曜日終了。 やっと休みだぜーーーーー。  しかし世間は不況ですね、トヨタの赤字が４５００億ですか。マツダも２５０億ですか。パナもけっこう最後まで隠してたけど３８００億。シャープが１０００億。 電機メーカーで唯一黒字の三菱電機が１００億。 世知辛いですなー。
