キーボード

最近よく押す

このボタン




いわゆるショートカットキーの話。

参考：ショートカットキー。
キーをうまく組み合わせていろんなことができる。
これができると便利、意外に知られてないこともあったりする。

なんか最近よく使うようになった気がするような。

昔から windows ボタン(キーボード左下にある windows の
マークのあるボタン)を押しながら D を押すってことはやってたけど。

参考：windows ボタン ＋ Ｄ ＝ ウィンドウ全部最小化

あと研究室で席たつ時とかに windowsボタン ＋ Ｌ (ログオフ)とか。


最近は仕事で ProgramData とか ProgramFiles を開くことが多いので
windows ボタン押しながら Ｅ　を押すことが増えた。

参考：windows ボタン ＋ Ｅ ＝ (マイ)コンピュータ開く


それとコマンドプロンプトとかレジストリエディタ開く時に
「ファイルを指定して実行」を起動させるのに
windows ボタン ＋ Ｒ とか。


暇な人はこれ読みながらやってみたらいいよ。

最近よくwindows ボタンを押してる。


あとはファイルを別のフォルダに移したり、名前変えたりしたあとに
「あ、ミスった」とか思った時に Ctrl ＋ Ｚ で元に戻したり。


最近先輩に教えてもらったのが間違えてよく分からんファイルを
ドラッグしてしまって放すに放せない時には キーボード左上の
Escape ボタンを押せばドラッグがキャンセルできる、とか。

あとコマンドプロンプトとかで使うのにフォルダとかファイルの
パスをクリップボードにコピーしたりするんだけど
パスをコピーしたいファイルとかを Shift を押しながら右クリックすると
メニューに「パスとしてコピー」ってのが出てくる、
これは Vista かららしい。


俺が知らんだけかもしれんけど先輩とかよく知ってる。
そういう情報どこで仕入れてくるんかね。

こういうの知ってるほうが仕事が速いっていうけども。
便利なものは覚えていきたいところ。
