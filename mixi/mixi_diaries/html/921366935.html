浸りはせんけど、あーとかおーとか思ったりな話。

今日午後に飛行機で東京に戻ってきまして。

戻ってくる時に実家の自分の部屋にあった CD ラックから
何枚か CD を持ってきた。

倉木麻衣のアルバムだとか、JUDY AND MARY のアルバムだとか。


バッハで借りてコピーした CD なんだけども。

参考；バッハ・広島にある TSUTAYA みたいなもん。

そういえば高校の頃は ipod みたいなもんはなくて
CD プレイヤーか MD ウォークマンの時代だったんよね。

それで CD プレイヤー派だった俺は借りてきては CD を作ってみたり、
高校の時好きだった人がゆず好きだってんでゆずの CD 作ってみたり。

テラナツカシスですな。

参考：テラ・・ギガの次の桁みたいなやつ。"めっちゃ・とても"の意。


そういえば日曜日にやってた２４時間テレビで
懐かしい人たちが出とったみたいね、
SPEED とか ZONE だとか。



レンタカーとか借りるときとか、車運転する時は BGM が大事。

今週も車に乗る予定があるので
また何枚か作っとくか。

今日で休み終了。
明日から会社。
まぁ３日行けばまた週末だからがんばろう。


夏はまだまだ終わりませんよ？
